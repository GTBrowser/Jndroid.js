HorizontalScrollView Class

Defined in: jndroid.widget.js:38

Layout container for a view hierarchy that can be scrolled by the user, allowing it to be larger than the physical display. A child that is often used is a LinearLayout in a horizontal orientation, presenting a horizontal array of top-level items that the user can scroll through.

HorizontalScrollView only supports horizontal scrolling. For vertical scrolling, use ScrollView.
